# eaaster-egg-hunt

Minimal Vite + React setup using TypeScript.

## Getting started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## GitHub Pages

The Vite base path is configured for repo-hosted Pages at `/eaaster-egg-hunt/`.

Deployment is handled by the GitHub Actions workflow in `.github/workflows/deploy.yml`.
In the repository settings, set Pages to deploy from GitHub Actions.
